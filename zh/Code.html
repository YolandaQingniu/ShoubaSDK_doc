基础公共事件

code 含义
0 无错误
1001 任务超时
1002 传入参数错误

扫描相关

code 含义
2001 蓝牙开关关闭
2002 定位开关关闭
2003 缺少粗略定位权限 android.permission.ACCESS_COARSE_LOCATION(Android 6~11 需要)
2004 缺少粗略定位权限 android.permission.ACCESS_COARSE_LOCATION(Android 6~11 需要)
2005 缺少蓝牙扫描权限 android.permission.BLUETOOTH_SCAN (Android12以上需要)
2006 缺少蓝牙连接权限 android.permission.BLUETOOTH_CONNECT (Android12以上需要)
2007 缺少允许设备被发现权限 android.permission.BLUETOOTH_ADVERTISE (Android12以上需要)
2008 不能扫描到任何设备
2009 此设备不支持蓝牙
2010 系统蓝牙状态未知（iOS特有）
2011 系统蓝牙重置中（iOS特有）
2012 系统蓝牙未授权 （iOS特有）
