QNSysBleStatusListener

系统蓝牙状态监听器

  • 用于监听回调系统蓝牙状态变化；

API

onSysBleStatus

系统蓝牙状态变化的回调

参数
名称 类型 说明
code int 系统蓝牙状态
0: 未知 1: 重置中 2: 不支持 3: 未授权 4: 已关闭 5: 已开启

QNSysBleStatus

名称 含义
QNSysBleStatusUnknown 未知
QNSysBleStatusResetting 重置中 (iOS特有)
QNSysBleStatusUnsupported 不支持
QNSysBleStatusUnauthorized 未授权 (iOS特有)
QNSysBleStatusPoweredOff 已关闭
QNSysBleStatusPoweredOn 可用，已开启
