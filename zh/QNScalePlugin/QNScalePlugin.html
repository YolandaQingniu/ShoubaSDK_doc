QNScalePlugin

QNScalePlugin

家用秤设备管理

API

setScalePlugin

设置家用秤管理类

参数
名称 类型 说明
plugin QNPlugin 基础管理类
返回值

类型：int 详情见状态码列表

setDeviceListener

设置设备监听器

参数
名称 类型 说明
deviceListener QNScaleDeviceListener 设备设备监听器

setStatusListener

设置设备状态监听器

参数
名称 类型 说明
statusListener QNScaleStatusListener 设备状态监听器

setDataListener

设置设备数据监听器

参数
名称 类型 说明
dataListener QNScaleDataListener 设备数据监听器

connectDevice

连接设备

参数
名称 类型 说明
device QNScaleDevice 家用秤设备
operate QNScaleOperate 设置项
返回值

类型：int 详情见状态码列表

cancelConnectDevice

断开设备连接

参数
名称 类型 说明
device QNScaleDevice 家用秤设备

setDeviceUnit

设置设备单位

参数
名称 类型 说明
device QNScaleDevice 家用秤设备
unit QNUnit 单位
返回值

类型：int 详情见状态码列表

getWeightLb

单位转换: kg转lb

参数
名称 类型 说明
weight String 测量数据返回的weight字符串
返回值
类型 说明
String 对应lb单位数值的字符串,一位有效数字

getWeightJin:

单位转换: kg转斤

参数
名称 类型 说明
weight String 测量数据返回的weight字符串
返回值
类型 说明
String 对应斤单位数值的字符串,一位有效数字

getWeightStLb:

单位转换: kg转StLb

参数
名称 类型 说明
weight String 测量数据返回的weight字符串
返回值
类型 说明
String [ ] [0] 对应St单位数值的字符串 ，整形 ； [1] 对应Lb单位数值的字符串， 一位有效数字

getWeightSt:

单位转换: kg转st

参数
名称 类型 说明
weight String 测量数据返回的weight字符串
返回值
类型 说明
String 对应st单位数值的字符串,两位有效数字
