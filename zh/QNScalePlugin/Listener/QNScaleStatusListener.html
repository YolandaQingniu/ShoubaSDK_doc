QNScaleStatusListener

QNScaleStatusListener

设备状态监听器，用于监听回调设备连接状态

API

onConnectedSuccess

家用秤设备连接成功

参数
名称 类型 说明
device QNScaleDevice 家用秤设备对象

onConnectFail

家用秤设备连接失败

参数
名称 类型 说明
code int 详情见状态码列表 状态码
device QNScaleDevice 家用秤设备对象

onReadyInteractResult

家用秤设备准备完成,可以开始设置交互

参数
名称 类型 说明
code int 详情见状态码列表 状态码
device QNScaleDevice 家用秤设备对象

onDisconnected

家用秤设备断开连接

参数
名称 类型 说明
device QNScaleDevice 家用秤设备对象
