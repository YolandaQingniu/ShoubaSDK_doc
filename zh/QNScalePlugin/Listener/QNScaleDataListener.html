QNScaleDataListener

QNScaleDataListener

设备数据监听器，用于监听回调设备测量数据；

API

onRealTimeWeight

设备实时体重回调

参数
名称 类型 说明
weight String 体重，单位kg
device QNScaleDevice 家用秤设备对象

onReceiveMeasureResult

设备测量完成结果数据回调

参数
名称 类型 说明
scaleData QNScaleData 测量数据结果
device QNScaleDevice 家用秤设备对象

onReceiveStoredData

设备收到存储数据列表回调

参数
名称 类型 说明
storedDataList Array <QNScaleData> 存储数据列表
device QNScaleDevice 家用秤设备对象
